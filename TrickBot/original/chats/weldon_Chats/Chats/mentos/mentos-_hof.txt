Readme.txt для инсталлятора VS_LLVM42.rar и интеграция обфускатора obfuscator-llvm-4.0 в Visual Studio
	0.	Загрузка и распаковка VS_LLVM42.rar		Пароль		dO2DdZOqT!{0b*XH*Y}*L>BbqVK<0E>*
	1.	Запускаем CMD под админом
	2.	cd "тот путь", где находится распакованный инсталлятор	VS_LLVM42.rar
	3.	install (существует 3 ключа: --full --lite --uninstall):
			install					интеграция в Visual Studio уже собранного обфускатора obfuscator-llvm-4.0, который должен находиться в C:\llvm4 (C:\llvm4\build\Release бинарники)
			install	--full			распаковка собранного обфускатора в C:\llvm4 и интеграция в Visual Studio (пригодится тем, кто захочет сам пересобрать обфускатор)
			install --lite			распаковка собранного обфускатора (только бинарники) в C:\llvm4\build\Release и интеграция в Visual Studio
			install --uninstall		деинсталяция из Visual Studio обфускатора obfuscator-llvm-4.0 (папка C:\llvm4 остается)
	4.	Настройка обфускатора под себя и компиляция своего C/C++ кода:
		- желательно выключить/минимизировать оптимизацию C/C++ (O2 -> Od это повысит эффективность обфусикации)
		- добавить в командную строку C/C++ ключи обфускации "-mllvm -fla -mllvm -sub -mllvm -bcf"
		- если при компиляции будет ругаться на ключи, их нужно отключить (в VS17/VS15 fpic в свойствах проекта "C/C++ Все параметры", в VS19 permissive- и/или diagnostics:column)
	5.	По сути это все. Можно все сделать руками, если что-то пошло не так, ReadmeEx.txt
